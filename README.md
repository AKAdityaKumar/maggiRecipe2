# Maggi Spice

1. Take a saucepan.
2. Add two spoons of oil on it.
3. Add chopped onion, green chillies and chopped garlic together.
4. Stir well.
5. Add salt if needed.
6. Pour water.
7. Add maggie masala
8. Wait till water boil.
9. Add maggie noodles
10. Leave it till dry.
11. Now enjoy.
